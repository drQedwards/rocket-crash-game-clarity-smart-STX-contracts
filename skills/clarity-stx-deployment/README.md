# Clarity STX deployment skill

This skill packages guidance, contract templates, deployment config, and starter
tests for Stacks Clarity projects using Clarinet and Hiro.

It focuses on neutral fair-flip and crash-style templates for STX, sBTC, and SIP-010
token wagers. Mainnet wagering deployments must use commit/reveal or signed
verifiable randomness and must complete the mainnet checklist.

Start with `SKILL.md`, then copy only the templates needed for the project.
